# AI Money Mentor 

## Problem
Many people don’t know how to manage money, save, or invest.

## Solution
This AI Money Mentor helps users:
- Analyze income & expenses
- Suggest savings & investment plan
- Provide financial advice using AI logic

## Features
- Money Health Score
- Savings suggestions
- Chat-based financial assistant

## Tech Stack
- Python (Flask)
- HTML
- JavaScript

## How to Run
1. Install Python
2. Run: python app.py
3. Open: http://127.0.0.1:5000/

## Impact
Helps users improve financial planning and savings habits.
